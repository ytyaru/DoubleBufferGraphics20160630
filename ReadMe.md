# このソフトウェアについて #

DoubleBufferGraphics20160630は私個人が学習目的で作成したソフトウェアです。

C++/win32のGDI+でダブルバッファリングしました。
BitmapとGraphicsの取得を1回のみにしたので![前回](https://github.com/ytyaru/HelloDoubleBuffer20160630)より高速に動作します。

# 開発環境 #

* Windows XP SP3
* C++/Win32
* Visual C++ 2010 Express
